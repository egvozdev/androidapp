# MyApp (пример)

Это минимальное Android-приложение на Kotlin (XML UI).

Как запустить:
1. Открой Android Studio.
2. Выбери "Open" и укажи корневую папку проекта (там, где settings.gradle).
3. Синхронизируй Gradle (обычно Android Studio сделает это автоматически).
4. Запусти приложение на эмуляторе или устройстве.

Дальше можно:
- Перенести на Jetpack Compose.
- Добавить навигацию, ViewModel, LiveData / Flow.
- Добавить сеть (Retrofit/Ktor) и хранение (Room / DataStore).
- Настроить CI (GitHub Actions).
